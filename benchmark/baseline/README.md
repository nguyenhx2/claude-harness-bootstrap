# baseline

`project-bootstrap`, the skill `harness-bootstrap` replaces, at the commit it was replaced.

It is vendored here so the "before" column in [`../RESULTS.md`](../RESULTS.md) can be recomputed by
anyone, on any machine, rather than trusted. `benchmark.py` reads this directory by default.

It is not installable and it is not maintained. It exists to be measured.
